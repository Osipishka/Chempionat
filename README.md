# Chempionat — Подготовка к чемпионату по веб-разработке

Репозиторий с задачами и проектами для подготовки к чемпионату по компетенции "Веб-разработка".

## Содержимое
- BestGames - Html CSS: Статическая страница о лучших играх (HTML + CSS + немного JS)
- BestGames - Html CSS_С: Исправленная/улучшенная версия
- FlaskApp: Простое веб-приложение на Python + Flask (backend)
- prilka_html: Дополнительные HTML-страницы/элементы

Технологии: HTML, CSS, JavaScript, Python (Flask)

Цель: практика верстки, базового backend и работы с HTTP.
